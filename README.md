## Overview
This project implements a console-based Tic Tac Toe game in Python.  
It was developed as an academic exercise to demonstrate core programming concepts, clean structure, and robust input handling.

The game supports two players and includes full validation of user input, accurate win detection, and tie handling.

## Features
- Console-based interactive gameplay
- Two-player mode
- Clear and readable board display
- Win detection (rows, columns, diagonals)
- Tie detection
- Input validation to prevent invalid moves or crashes
- Option to restart the game after completion
